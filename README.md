# Cut the Rope: Multiple Treat

## About

**Cut the Rope: Multiple Treat** is a fork of _Cut the Rope: DX_ based on CtR 3DS port called _Triple Treat_ which will add 2 more games to Cut the Rope: _Experiments_ and _Time Travel_. Also you will can add your own map packs. As for now, this project didn't even started, because I just made a fork, but there will be stuff soon! See you in Multiple Treat!
